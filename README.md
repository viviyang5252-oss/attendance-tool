# 无尽冬日联盟考勤工具 · 使用手册

> 面向联盟管理员的日常使用指南

---

## 一、工具网址

https://www.wjdrtool.icu/

建议收藏或添加到手机主屏幕，方便日常使用。

---

## 二、首次使用：配置云同步（必做）

**只有配置了云同步，手机和电脑才能共享同一份考勤数据。**

### 操作步骤

1. 打开考勤工具网址
2. 点击页面**右上角**的 **⚙ 未配置**（或 **⚙ 已同步**）
3. 在弹出的设置窗口中：
   - **GitHub Personal Access Token**：粘贴负责人发给您的 Token（**请勿泄露给他人**）
   - **Gist ID**：
4. 点击 **保存并同步**
5. 右上角显示 **✓ 已同步** 即配置成功

### 获取 Token

- **方式一**：联系联盟负责人，由负责人私密发送 Token 给您
- **方式二**：若您有 GitHub 账号，可自行生成 Token（需勾选 `gist` 权限）

> ⚠️ Token 相当于密码，请勿在群聊、截图或公开文档中分享。

## 三、日常使用流程

### 步骤 1：选择 / 新建活动

- **日期**：选择活动日期（如 2026-03-11）
- **活动名称**：填写活动名（如 堡垒战、联盟战）
- 点击 **新建**
- 或从 **加载已有活动** 下拉框选择历史活动

---

### 步骤 2：维护应到名单

应到名单只需维护一次，之后可反复使用。支持三种方式：

| 方式 | 操作 |
|------|------|
| **手动粘贴** | 每行一个名字，粘贴到文本框，点 **保存名单** |
| **导入文件** | 上传 Excel 或 CSV，第一列为名字 |
| **截图识别** | 上传联盟成员列表截图，识别后编辑再保存 |

---

### 步骤 3：上传打卡截图并识别

1. 完成步骤 1，选择或新建活动
2. 点击 **选择战报截图**，可多选
3. 点击 **开始识别**
4. 识别完成后，在表格中检查名字：
   - **✓ 已匹配** / **≈ 模糊匹配**：无需修改
   - **未匹配，请修改名字**：点击名字直接修改
   - 误识别的行可点 **删除** 或使用 **一键清空** 后重新识别

---

### 步骤 4：查看结果并导出

- 查看 **应到 / 实到 / 缺勤 / 未匹配** 统计
- 点击 **导出 Excel (.xlsx)** 下载考勤表
- 导出的 Excel 中：实到打 ✓，缺勤留空，未匹配单独列出

---

## 四、可选设置（点右上角 ⚙ 进入）

| 设置项 | 说明 |
|--------|------|
| **百度 OCR** | 配置后识别速度从数十秒缩短到 1–3 秒，需自行注册百度智能云 |
| **自定义过滤词** | 识别后自动过滤掉这些词（如 删除、邮件），每行一个 |

## 五、常见问题

**Q：换了手机/电脑，数据还在吗？**  
A：在。只要在新设备上重新配置一次 Token 和 Gist ID，数据会自动同步过来。

**Q：识别出来的名字有错字怎么办？**  
A：在打卡名单表格里直接点击名字修改，修改后统计会自动更新。

**Q：识别到「删除」「邮件」等无关文字怎么办？**  
A：点右上角 ⚙ → 在「自定义过滤词」里添加这些词，每行一个，保存即可。

**Q：同步失败 / 显示 ✗ 同步失败？**  
A：Token 可能已过期，请联系负责人重新获取 Token，或自行在 GitHub 重新生成后更新设置。

---

## 六、快捷操作

- **一键清空**：清空当前活动的打卡名单，可重新识别
- **手动添加**：在打卡名单中手动添加漏记的名字
- **立即同步**：在历史记录区域点击，可手动触发一次数据同步
